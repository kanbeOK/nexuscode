# Changelog

## [1.0.0] - 2026-06-13

### Added
- 11 specialized AI agents
- Band-only communication (no orchestrator)
- Adversarial code review (Reviewer + RedTeamer)
- Evidence verification (Verifier)
- Persistent memory with search
- SHA-256 tamper-evident audit trail
- Veto loop for proposal rejection
- Human approval gates
- LoopAgent and SequentialAgent patterns
- Pydantic structured outputs
- Web dashboard (Next.js)
- REST API
- WebSocket real-time updates
- Docker support

### Patterns Implemented
- Band-Only Communication (WarRoom)
- Adversarial Review (AEGIS)
- Persistent Memory (Band Memory)
- Veto Loop (Band Decision Desk)
- Human Gates (Balai)
- Virtual Company (ChatDev)
- LoopAgent/SequentialAgent (Google Codelabs)
- Structured Output (Google Codelabs)

- NexusPlanner: Requirements analysis
