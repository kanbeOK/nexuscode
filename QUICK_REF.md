# NexusCode - Quick Reference Card

## For Hackathon Judges

### What is NexusCode?
A **Band-powered multi-agent system** where 11 specialized AI agents collaborate through Band as their **only** communication channel to automate software development.

### Key Innovation
**Band-Only Communication** — No orchestrator, no message bus, no shared memory. Every interaction is a Band @mention. The workflow **emerges** from who addresses whom.

### 11 Agents
1. **NexusPlanner** — Requirements & user stories
2. **NexusArchitect** — System design & APIs
3. **NexusDeveloper** — Code implementation
4. **NexusReviewer** — Quality review (cooperative)
5. **NexusRedTeamer** — Security review (adversarial)
6. **NexusVerifier** — Evidence verification
7. **NexusQA** — Testing & quality
8. **NexusDevOps** — Deployment & CI/CD
9. **NexusScribe** — Documentation
10. **NexusAdjudicator** — Conflict resolution
11. **NexusHumanGate** — Human approval enforcement
